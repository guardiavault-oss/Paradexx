## App Review Notes

### Apple App Store Review

```
Paradox is a non-custodial cryptocurrency wallet. Users maintain full control of their private keys which are stored securely on their device using iOS Keychain.

The app does NOT:
- Process fiat currency payments
- Offer financial advice
- Facilitate gambling or betting
- Allow users to purchase cryptocurrency directly

For testing, you can create a new wallet with these steps:
1. Open the app
2. Select "Easy Mode" for seed phrase wallet
3. Create a password
4. The wallet will be ready to use on testnet

Contact: review@paradox.io
```

### Google Play Review

```
Paradox is a self-custody cryptocurrency wallet. Users control their own private keys.

The app requires these permissions:
- CAMERA: To scan QR codes for wallet addresses
- BIOMETRIC: For secure wallet unlock
- INTERNET: To interact with blockchain networks
- NOTIFICATIONS: For transaction alerts

The app does not process payments or facilitate gambling.

Contact: review@paradox.io
```
