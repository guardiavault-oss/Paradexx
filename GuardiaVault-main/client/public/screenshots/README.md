# PWA Screenshots

This directory should contain screenshots of your app for PWA store listings.

## Required Screenshots

- `desktop-1.png` - 1280x720 pixels (wide format for desktop)
- `mobile-1.png` - 750x1334 pixels (narrow format for mobile)

## Screenshot Guidelines

- **Desktop**: Should showcase the main features of your app in landscape orientation
- **Mobile**: Should showcase the mobile experience in portrait orientation
- **Format**: PNG
- **Content**: Should highlight key features, UI, and functionality
- **Quality**: High resolution, clear and professional

## Taking Screenshots

1. **Desktop**: Use browser dev tools to set viewport to 1280x720, take screenshot
2. **Mobile**: Use browser dev tools to set viewport to 375x812 (iPhone X), take screenshot

## Tools

- Browser DevTools (Chrome/Edge/Firefox)
- [Responsively App](https://responsively.app/)
- [BrowserStack](https://www.browserstack.com/)

## Temporary Solution

You can use placeholder images or existing dashboard screenshots from your app.
